# Rock, Paper, Scissors With Python

A simple command-line Rock, Paper, Scissors game built in Python.
The program allows the user to play against the computer by selecting Rock, Paper, or Scissor, with results determined based on standard game rules.

Key features:

Randomized computer choice using the random module

Clear user input and result display

Structured logic for win, lose, and draw conditions

Beginner-friendly and easy to extend

This project was created to practice Python fundamentals such as conditionals, loops, and dictionary handling.
